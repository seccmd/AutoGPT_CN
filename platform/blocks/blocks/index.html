    AutoGPT 模块概览¶

    AutoGPT 采用模块化方法，通过不同的“模块”来处理各种任务。这些模块是 AutoGPT 工作流的基础构建块，允许用户通过组合简单、专门的组件来创建复杂的自动化流程。

    以下是所有可用模块的完整列表，按其主要功能分类。点击任何模块名称可查看其详细文档。

    基本操作¶

    模块名称 描述
    存储值 存储并转发一个值
    打印到控制台 将文本输出到控制台以进行调试
    在字典中查找 在字典或列表中查找值
    代理输入 在工作流中接受用户输入
    代理输出 记录并格式化工作流结果
    添加到字典 向字典中添加新的键值对
    添加到列表 向列表中添加新条目
    便签 在工作流中显示便签

    数据处理¶

    模块名称 描述
    读取 CSV 处理并提取 CSV 文件中的数据
    数据采样 使用各种采样方法选择数据的子集

    文本处理¶

    模块名称 描述
    匹配文本模式 检查文本是否匹配指定模式
    提取文本信息 使用模式从文本中提取特定信息
    填充文本模板 用提供的值填充模板
    合并文本 将多个文本输入合并为一个
    文本解码器 将编码文本转换为可读格式

    AI 和语言模型¶

    模块名称 描述
    AI 结构化响应生成器 使用 LLM 生成结构化响应
    AI 文本生成器 使用 LLM 生成文本响应
    AI 文本摘要器 使用 LLM 对长文本进行摘要
    AI 对话 使用 LLM 进行多轮对话
    AI 列表生成器 使用 LLM 根据提示生成列表

    网络和 API 交互¶

    模块名称 描述
    发送网络请求 向指定网址发送 HTTP 请求
    读取 RSS 订阅 检索并处理 RSS 订阅中的条目
    获取天气信息 获取指定位置的当前天气数据
    Google 地图搜索 使用 Google 地图 API 搜索本地商家

    社交媒体和内容¶

    模块名称 描述
    获取 Reddit 帖子 从指定子版块中检索帖子
    发布 Reddit 评论 在 Reddit 上发布评论
    发布到 Medium 直接将内容发布到 Medium
    读取 Discord 消息 从 Discord 频道中检索消息
    发送 Discord 消息 向 Discord 频道发送消息

    搜索和信息检索¶

    模块名称 描述
    获取维基百科摘要 从维基百科获取主题摘要
    搜索网络 执行网络搜索并返回结果
    提取网站内容 检索并提取网站内容

    时间和日期¶

    模块名称 描述
    获取当前时间 提供当前时间
    获取当前日期 提供当前日期
    获取当前日期和时间 提供当前日期和时间
    倒计时器 作为倒计时器使用

    数学和计算¶

    模块名称 描述
    计算器 执行基本数学运算
    计数项目 计算集合中的项目数量

    媒体生成¶

    模块名称 描述
    Ideogram 模型 根据文本提示生成图像
    创建会说话的虚拟形象视频 创建带有会说话的虚拟形象的视频
    Unreal 文本转语音 使用 Unreal Speech API 将文本转换为语音
    AI 短视频生成器 使用 AI 生成短视频
    Replicate Flux 高级模型 使用 Replicate 的 Flux 模型创建图像

    其他¶

    模块名称 描述
    转录 YouTube 视频 转录 YouTube 视频中的音频
    发送电子邮件 使用 SMTP 发送电子邮件
    条件模块 评估工作流分支的条件
    逐步遍历项目 遍历列表或字典

    Google 服务¶

    模块名称 描述
    Gmail 读取 从 Gmail 账户中检索并读取电子邮件
    Gmail 发送 使用 Gmail 账户发送电子邮件
    Gmail 列出标签 从 Gmail 账户中检索所有标签
    Gmail 添加标签 向 Gmail 账户中的特定电子邮件添加标签
    Gmail 移除标签 从 Gmail 账户中的特定电子邮件中移除标签
    Google Sheets 读取 从 Google Sheets 电子表格中读取数据
    Google Sheets 写入 向 Google Sheets 电子表格中写入数据
    Google 地图搜索 使用 Google 地图 API 搜索本地商家

    GitHub 集成¶

    模块名称 描述
    GitHub 评论 在 GitHub 问题或拉取请求上发布评论
    GitHub 创建问题 在 GitHub 仓库中创建新问题
    GitHub 读取问题 检索特定 GitHub 问题的信息
    GitHub 列出问题 从 GitHub 仓库中检索问题列表
    GitHub 添加标签 向 GitHub 问题或拉取请求添加标签
    GitHub 移除标签 从 GitHub 问题或拉取请求中移除标签
    GitHub 分配问题 将用户分配给 GitHub 问题
    GitHub 列出标签 检索并列出指定 GitHub 仓库的所有标签
    GitHub 列出分支 检索并列出指定 GitHub 仓库的所有分支
    GitHub 列出讨论 检索并列出指定 GitHub 仓库的最近讨论
    GitHub 创建分支 在 GitHub 仓库中创建新分支
    GitHub 删除分支 从 GitHub 仓库中删除指定分支
    GitHub 列出拉取请求 从指定 GitHub 仓库中检索拉取请求列表
    GitHub 创建拉取请求 在指定 GitHub 仓库中创建新拉取请求
    GitHub 读取拉取请求 检索特定 GitHub 拉取请求的详细信息
    GitHub 分配 PR 审查者 将审查者分配给特定 GitHub 拉取请求
    GitHub 取消分配 PR 审查者 从特定 GitHub 拉取请求中移除分配的审查者
    GitHub 列出 PR 审查者 检索特定 GitHub 拉取请求的所有分配审查者列表

    Twitter 集成¶

    模块名称 描述
    Twitter 发布推文 在 Twitter 上创建包含文本内容和可选附件（包括媒体、投票、引用或深层链接）的推文
    Twitter 删除推文 使用推文 ID 删除指定推文
    Twitter 搜索最近推文 搜索符合指定条件的推文，并提供过滤和分页选项
    Twitter 获取引用推文 获取引用指定推文 ID 的推文，并提供分页和过滤选项
    Twitter 转发 使用推文 ID 创建指定推文的转发
    Twitter 移除转发 移除指定推文的现有转发
    Twitter 获取转发者 获取转发指定推文的用户列表，并提供分页和过滤选项
    Twitter 获取用户提及 获取提及特定用户的推文，使用其用户 ID
    Twitter 获取主页时间线 获取认证用户及其关注账户的最近推文和转发
    Twitter 获取用户 获取单个 Twitter 用户的详细资料信息
    Twitter 获取多个用户 获取多个 Twitter 用户的资料信息（最多 100 个）
    Twitter 搜索空间 根据标题关键词搜索 Twitter 空间，并提供状态过滤
    Twitter 获取空间 通过空间 ID 或创建者 ID 获取多个 Twitter 空间的信息
    Twitter 获取空间详情 获取单个 Twitter 空间的详细信息
    Twitter 获取空间推文 获取在 Twitter 空间会话期间分享的推文
    Twitter 关注列表 使用列表 ID 关注 Twitter 列表
    Twitter 取消关注列表 取消关注之前关注的 Twitter 列表
    Twitter 获取列表 获取特定 Twitter 列表的详细信息
    Twitter 获取拥有的列表 获取指定用户拥有的所有 Twitter 列表
    Twitter 获取列表成员 获取指定 Twitter 列表的成员信息
    Twitter 添加列表成员 将指定用户添加为 Twitter 列表的成员
    Twitter 移除列表成员 从 Twitter 列表中移除指定用户
    Twitter 获取列表推文 获取指定 Twitter 列表中发布的推文
    Twitter 创建列表 使用指定名称和设置创建新的 Twitter 列表
    Twitter 更新列表 更新现有 Twitter 列表的名称和/或描述
    Twitter 删除列表 删除指定的 Twitter 列表
    Twitter 固定列表 将 Twitter 列表固定在列表顶部
    Twitter 取消固定列表 从固定列表中移除 Twitter 列表
    Twitter 获取固定列表 获取当前固定的所有 Twitter 列表
    Twitter 列表获取关注者 正在开发中... 获取指定 Twitter 列表的所有关注者
    Twitter 获取关注的列表 正在开发中... 获取用户关注的所有列表
    Twitter 获取私信事件 正在开发中... 检索用户的私信事件
    Twitter 发送私信 正在开发中... 向指定用户发送私信
    Twitter 创建私信会话 正在开发中... 创建新的私信会话

    Todoist 集成¶

    模块名称 描述
    Todoist 创建标签 在 Todoist 中创建新标签
    Todoist 列出标签 从 Todoist 中检索所有个人标签
    Todoist 获取标签 通过 ID 检索特定标签
    Todoist 创建任务 在 Todoist 中创建新任务
    Todoist 获取任务 从 Todoist 中检索活动任务
    Todoist 更新任务 更新现有任务
    Todoist 关闭任务 完成/关闭任务
    Todoist 重新打开任务 重新打开已完成的任务
    Todoist 删除任务 永久删除任务
    Todoist 列出项目 从 Todoist 中检索所有项目
    Todoist 创建项目 在 Todoist 中创建新项目
    Todoist 获取项目 检索特定项目的详细信息
    Todoist 更新项目 更新现有项目
    Todoist 删除项目 删除项目及其内容
    Todoist 列出协作者 检索项目上的协作者
    Todoist 列出部分 从 Todoist 中检索部分
    Todoist 获取部分 检索特定部分的详细信息
    Todoist 删除部分 删除部分及其任务
    Todoist 创建评论 在任务或项目上创建新评论
    Todoist 获取评论 检索任务或项目的所有评论
    Todoist 获取评论 通过 ID 检索特定评论
    Todoist 更新评论 更新现有评论
    Todoist 删除评论 删除评论

    此完整列表涵盖了 AutoGPT 中所有可用的模块。每个模块都设计用于执行特定任务，并且可以组合起来创建强大的自动化工作流。有关每个模块的更多详细信息，请点击其名称查看完整文档。
