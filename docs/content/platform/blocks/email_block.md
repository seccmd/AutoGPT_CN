# 发送电子邮件

## 功能概述
“发送电子邮件”模块是一个利用SMTP（简单邮件传输协议）凭据发送电子邮件的工具。

## 功能描述
该模块允许用户向指定收件人发送带有自定义主题和正文的电子邮件。它通过提供的SMTP凭据连接到电子邮件服务器并发送消息。

## 工作原理
该模块接收收件人的电子邮件地址、主题和邮件正文作为输入。同时，它还需要SMTP凭据，包括服务器地址、端口、用户名和密码。模块随后连接到指定的SMTP服务器，使用提供的凭据进行身份验证，并发送电子邮件。在尝试发送电子邮件后，它会报告操作是否成功或是否发生了错误。

## 输入参数
| 输入项 | 描述 |
|-------|-------------|
| 收件人邮箱 | 收件人的电子邮件地址 |
| 主题 | 电子邮件的主题行 |
| 正文 | 电子邮件的主要内容 |
| SMTP凭据 | 用于身份验证的服务器、端口、用户名和密码 |

### SMTP凭据详情
| 凭据项 | 描述 | 默认值 |
|------------|-------------|---------|
| SMTP服务器 | SMTP服务器的地址 | smtp.gmail.com |
| SMTP端口 | SMTP服务器的端口号 | 25 |
| SMTP用户名 | 用于SMTP服务器身份验证的用户名 | - |
| SMTP密码 | 用于SMTP服务器身份验证的密码 | - |

## 输出结果
| 输出项 | 描述 |
|--------|-------------|
| 状态 | 指示电子邮件是否成功发送的消息 |
| 错误 | 如果电子邮件发送失败，此输出提供有关发生错误的详细信息 |

## 应用场景示例
该模块可用于自动化客户支持系统。当客户通过网站提交支持票证时，“发送电子邮件”模块可以自动向客户发送确认邮件，确认收到他们的请求，并为他们提供用于未来参考的票证编号。
