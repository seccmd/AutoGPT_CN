## **先决条件**
* 本地安装的AutoGPT
* 访问本地托管者的市场

## **逐步流程**

### **1. 访问市场**
* 导航到市场部分
    * 注意：界面将与云托管平台不同

### **2. 下载代理**
1. 浏览并选择您所需的代理
2. 点击代理以查看详细信息
3. 查找并点击“下载代理”按钮
4. 代理文件将保存到您的计算机

### **3. 导入代理**
1. 返回到监控选项卡
2. 找到“创建”旁边的下拉菜单
3. 选择“从文件导入”
4. 在文件选择对话框中：
    * 导航到您下载的代理文件
    * 选择文件
    * 点击“打开”
5. 点击“导入并编辑”

### **4. 验证导入**
* 代理现在应该出现在您的构建器中
* 您可以开始编辑或使用导入的代理

## **重要注意事项**
* 市场界面在本地托管和云托管版本之间有所不同
* 确保您遵循本地托管的特定步骤
* 成功导入后，代理将在您的构建器中可用
